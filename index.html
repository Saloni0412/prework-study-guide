HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • elements - entered in < > brackets
  • variables - are mentioned below element, eg. href, alt, width
  • header - contains introductory part and is at the top
  • footer - conatins author, copyright, sitemap etc and is at the bottom
  • HEAD - Metadata - info about page used by browser - user cannot see
  • HEAD - Title - title of the page - user can see
  • h1, h2, h3 ... h6 - levels of heading, h1 being the highest.
  • use "P" to add paragraph or blocks of test.
  • use "ul" for unordered list
  • use "ol" for ordered list
  • use "li" for listing the item under "ol" or "ul"
  • use "img" to insert image, which has diffrent attributes
  • src - location of image, width, height , alt - is used to give brief description of image and helps improve search engine ranking
  • use "br to break line or creates line of spaces"
  • use "a" to create link to same of different webpage
  • href - attaches URL or links

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations - CSS property we want to apply and the value of the property.
  • * - to apply a rule to all the elements visible on the page. * - to apply a rule to all the elements visible on the page.
  • we can put two selector together if we want same styling for them using a comma (,)
  • The "img" selector assigns property values to any image element in the markup.
  • "link" - use element to connect css style to html file

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -a: commit all files in working directory
  • commit -m (text): associate msg with commit, short and descriptive
  • pull origin main: to check if current branch is up to date
  • push origin (folder name): pushes all files to changes
